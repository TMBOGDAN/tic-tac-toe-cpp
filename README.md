# Tic Tac Toe – C++

This is a simple console-based **Tic Tac Toe (X and O)** game implemented in **C++**.
The game is designed for two players and includes score tracking and the option to replay multiple rounds.

## Features
- Two-player gameplay (X vs O)
- 3x3 game board
- Win detection (rows, columns, diagonals)
- Draw (tie) detection
- Score system for both players
- Replay option after each match
- Modular code using multiple functions

## Technologies Used
- C++
- Standard Library (`iostream`)

## Files
- `tictactoe.cpp` – main source file containing the game logic

## How to Run
Compile and run the program using a C++ compiler (e.g. g++):

```bash
g++ tictactoe.cpp -o tictactoe
./tictactoe
